# 📚 Tugas 8 - Templating Admin Dashboard

Project ini merupakan implementasi **templating AdminLTE 4** menggunakan **PHP Native**. Pada tugas ini dilakukan pemisahan template menjadi beberapa bagian agar kode lebih terstruktur dan mudah dikelola.

## 👨‍💻 Teknologi

- PHP Native
- AdminLTE 4
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
